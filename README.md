# 🩸 Blood Bank Management System

## 💡 Description
A Java Swing-based desktop application for managing blood donors.

## 🚀 Features
- Add donor
- View donors
- Search by blood group
- Edit donor details
- Save data using file handling

## 💻 Tech Used
- Java
- Swing (GUI)
- File Handling (Serialization)
